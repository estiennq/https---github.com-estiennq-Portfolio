J'ai crée un assistant IA capable de résoudre des problèmes de cybersécurité de type CTF. Le modèle d'IA est Ollama Mistral, un modèle léger et pouvant tourner sans cpu, permettant de le faire tourner sur la plupart des ordinateurs.

L'assistant fonctionne selon 2 modes :

  • Mode Rapport

    • Un script bash récupère le contenu du dossier CTF crée un prompt et l'envoie à l'IA, qui répond par un rapport détaillé.

  • Mode autonome

    • Avec l'aide de Langchain, l'IA est dôté d'outils lui permettant d'explorer l'arborescence du défi CTF et d'agir sur celui-ci. Elle peut ainsi exécuter du code en fonction de ce qu'elle veut accomplir.

Compétence développées

  • Code IA

    • Apprentissage de l'interaction avec IA avec Python.

  • Framework IA (Langchain)

    • Apprentissage de Langchain pour donner des outils à l'IA lui permettant d'interagir avec l'environnement.
